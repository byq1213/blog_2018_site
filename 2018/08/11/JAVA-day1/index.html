Java 安装准备工作

2018-08-10

常用DOS 命令

命令 解释
D: 回车 盘符切换
dir (directory) 列出当前目录下的文件以及文件夹
md (make directory) 删除目录
cd (change directory) 改变指定目录（进入指定目录）
cd.. 返回上一级目录
cd \ 返回到根目录
del (delete) 删除文件，（*.txt 删除所有txt 文件）
cls (clear sreen) 清屏
exit 退出DOS 命令行

Java 语言平台版本

  • J2SE(Java 2 Paltform Standard Edition) 标准版
    • 为开发普通桌面和商务应用程序提供的解决方案
    • 该技术体系为其他两者的基础
  • J2ME(Java 2 Platform Micro Edition) 小型版
    • 开发电子消费产品和嵌入式设备提供的解决方案
  • J2EE(Java 2 Platform Enterprise Edition) 企业版
    • 为开发企业环境下的应用程序提供的一套解决方案
    • 该技术体系中包含 Servlet、Jsp等，主要针对于web程序开发

JVM & JRE & JDK

  • JVM(Java Virtual Machine) Java 运行所需虚拟机
  • JRE(Java Runtime Environment) Java 运行环境
    • 包括 JVM 和Java所需的核心类库等，可运行一个开发好的Java 程序
  • JDK(Java Development Kit) Java 开发工具包
    • 提供给Java 开发人员使用，包含Java开发工具 & JRE ，所以安装了JDK，就不用单独安装 JRE。
    • 简单而言：使用JDK开发完成的Java程序，交给JRE运行

JDK 安装路径下的目录结构（Java 10.0之前）

目录 解释
bin 存放可执行程序 如：javac.exe 编译器、java.exe 运行工具
db JavaDB小型数据库
jre 运行环境根目录
include 由于JDK是通过C和C++实现的，因此启动时需要引入一些C语言的头文件，该目录就是用于存放这些头文件的。
lib library的缩写，意为 Java 类库或库文件，是开发工具使用的归档包文件。
src.zip src文件夹的压缩文件，存放JDK源代码

环境变量配置

变量名 变量
JAVA_HOME ~安装目录
PATH + %JAVA_HOME%/bin;
classpath .;

标识符命名规则

类型 要求
包名（文件夹名） 区别相同类名的问题，全部小写，一般为项目域名反写 com.baidu.project
类名 单个单词，首字母大写；多个单词拼接，每个单词首字母大写，（大驼峰命名） HelloWorld、Fuck
方法名、变量名 单个单词，全部小写；多个单词，第二个单词首字母开始大写 helloWorld、fuckYou
常量 单个单词，全部大写；多个单词，也全部大写，每个单词中间_分隔 URL、SET_TOKEN
